CalFin Indonesia

Kebijakan Privasi

Penjelasan ringkas tentang data yang diproses saat Anda menggunakan CalFin.

Berlaku mulai: 7 Maret 2026

Ringkasan

  • CalFin tidak mewajibkan akun pengguna.
  • CalFin tidak memiliki backend untuk menyimpan data pengguna.
  • Sebagian besar perhitungan dilakukan langsung di perangkat.
  • CalFin menggunakan Google AdMob untuk menampilkan iklan banner Android.
  • CalFin dapat mengambil data referensi publik dari situs resmi bank.
  • CalFin dapat membuka tautan eksternal yang Anda pilih sendiri.

Data yang Diproses di Perangkat

CalFin memproses input simulasi yang Anda masukkan, seperti nominal pinjaman, tenor, bunga, target tabungan, nominal deposito, dan frekuensi transfer.

Pada fitur prospek KPR, CalFin juga dapat memproses data yang Anda masukkan sendiri, seperti nama, nomor telepon, kota, penghasilan, dan catatan.

Penyimpanan Data

CalFin tidak mengunggah data simulasi atau data prospek ke server CalFin. Data diproses di memori aplikasi saat aplikasi digunakan. Jika Anda menyalin data ke clipboard, data tersebut akan berada di clipboard perangkat sampai Anda menggantinya.

Iklan dan SDK Pihak Ketiga

CalFin menggunakan Google Mobile Ads SDK melalui Google AdMob untuk memuat iklan banner di Android. SDK ini dapat memproses identifier iklan atau identifier perangkat serupa, informasi aplikasi dan perangkat, alamat IP, informasi jaringan, sinyal performa dasar, serta interaksi dasar dengan iklan untuk penyajian iklan dan pencegahan fraud.

Akses Internet dan Referensi Publik

CalFin menggunakan internet untuk mengambil referensi umum dari sumber resmi bank , memuat iklan banner dari Google AdMob, dan untuk membuka tautan eksternal yang Anda pilih sendiri. Jika internet tidak tersedia, aplikasi memakai data lokal atau fallback agar simulasi tetap berjalan.

Tautan dan Layanan Pihak Ketiga

CalFin dapat menggunakan atau membuka layanan pihak ketiga seperti Google AdMob, situs bank, Google Maps, WhatsApp, Google Sheets, atau browser perangkat. Setelah Anda keluar dari CalFin dan membuka layanan tersebut, penggunaan data Anda akan mengikuti kebijakan privasi layanan pihak ketiga masing-masing.

Izin Aplikasi

  • INTERNET, untuk mengambil referensi publik dan membuka layanan online.
  • ACCESS_NETWORK_STATE, untuk memeriksa kondisi jaringan dan fallback offline.

Pembagian Data

CalFin tidak menjual data pribadi. Data hanya dapat keluar dari aplikasi jika Anda sendiri memilih tindakan seperti menyalin ke clipboard, membuka WhatsApp, membuka Google Sheets, atau membuka tautan eksternal. Selain itu, SDK iklan pihak ketiga dapat menerima data teknis yang diperlukan untuk penyajian iklan dan pencegahan fraud.

Kontak

Untuk pertanyaan privasi, gunakan alamat email pengembang yang tercantum pada listing resmi Google Play CalFin saat aplikasi dipublikasikan.

Dokumen halaman ini diselaraskan dengan file PRIVACY_POLICY.md di repo CalFin dan dapat dipakai sebagai dasar URL publik untuk Google Play.
